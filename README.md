📖 “蒸蒸日上的胜率” 机器人使用指南
欢迎使用我们的国战胜率记录助手！以下是所有可用指令的详细说明。 (注：所有指令均以 # 开头，支持中文或拼音输入)

1. 📝 记录战绩 (#add)
一局游戏结束后，请直接复制粘贴玩家信息。

格式：玩家名 主将 副将 [胜者标记]

胜者标记：在获胜者的行尾加上 💅 (指甲油)、🏆、👑 或 win 均可识别。

智能识别：机器人能自动识别武将昵称（如“大宝”、“香香”、“双头”）。

示例：

Plaintext

#add
殊 孙尚香 丁奉💅
MU 刘备 魏延
DH 狗货 刚烈
JX 黄忠 祝融
ZS 邹氏 孔融
Joyce 大宝 凌统💅
Wu 卧龙 庞统
Jason 司马 邓艾
2. 📅 查看今日战报 (#date)
生成一张包含今日所有对局的 Excel 风格图片，直观展示每个人的武将组合和胜负情况。

查看今天：

Plaintext

#date
查看指定日期：

Plaintext

#date 2025-12-09
3. 🔍 查看单局详情 (#game)
如果你想回看某一局的具体配置。

查看最新一局：

Plaintext

#game last
查看指定局号（例如第 5 局）：

Plaintext

#game 5
4. 📊 查看胜率排行榜 (#rates)
展示所有玩家的“胜场/总场”以及“胜率”排名。

指令：

Plaintext

#rates
5. 📧 导出 Excel 到邮箱 (#email)
将所有历史对局记录生成 Excel 表格，发送到你的邮箱。表格中获胜的武将会高亮显示。

指令：

Plaintext

#email yourname@gmail.com
6. ✏️ 修改与删除 (管理功能)
如果记录记错了，可以使用以下指令进行修正。机器人会弹出确认按钮，防止手滑。

修改某局信息： 先输入 #update 局号，换行后输入新的完整对局信息。

Plaintext

#update 10
殊 孙尚香 丁奉
MU 刘备 赵云 💅
(输入其余玩家...)
删除某局信息：

Plaintext

#remove 10
